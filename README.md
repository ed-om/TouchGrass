# TouchGrass

An Android application built with **Kotlin** that encourages users to reduce screen time by interacting with nature.

## Features
*   **Nature Verification:** Uses **Google ML Kit** to identify and verify if the user is actually looking at grass or greenery.
*   **High Performance:** Optimized for smooth Android performance.
*   **Privacy-Focused:** Screen time tracking and image labeling done on-device.

## Tech Stack
*   **Language:** Kotlin
*   **Platform:** Android Studio
*   **AI/ML:** Google ML Kit (Image Labeling)

## How to Run
1. Clone this repository.
2. Open in Android Studio.
3. Build and run on an Android device or emulator.
